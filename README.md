# Разработка информационной системы для завода по производству хлебобулочных изделий в рамках демонстрационного экзамена

Информационная система для завода по производству хлебобулочных изделий - предназначена для управления протекающими на заводе процессами, непосредственно связанные с данными пользователей.  
Система предполагает основной функционал работы с данными пользователей. Гибкую форму авторизации со Смарт-капчей.

## Необходимые условия
- .NET 4.8 - Основная среда выполнения
- SQL SERVER - Провайдер базы данных
- SSMS - Средство управления SQL SERVER с помощью пользовательского интерфейса
